# 🚀 HackerRank Landing Page Clone

> Went to solve HackerRank questions, made a landing page clone instead.

A pixel-perfect, fully responsive clone of the HackerRank landing page built from scratch using **React**, **Vite**, and **Tailwind CSS v4**. This project focuses on recreating the original UI while implementing custom interactive components without relying on heavy third-party libraries.

---

## ✨ Features

- 📱 Fully Responsive Design
- 🎨 Pixel-perfect HackerRank-inspired UI
- 🍔 Custom Mobile Navigation Menu
- 🎞️ Infinite Auto-playing Testimonial Carousel
- 🎥 HTML5 Video Integration
- ⚡ Built with React 19 + Vite
- 🎯 Tailwind CSS v4 Styling
- 🧩 Reusable Component-Based Architecture
- 🌙 HackerRank Dark Theme Recreation

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/ramanbhatia3/HackerRank-Clone.git
```

### Navigate into the Project

```bash
cd HackerRank-Clone
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Visit:

```
http://localhost:5173
```

---

## 📦 Build for Production

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
